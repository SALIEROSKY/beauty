Bienvenido

  • El operador igualdad (==) se utiliza para comparar si dos valores son iguales.
  • El operador desigualdad (!=) se utiliza para comparar si dos valores son diferentes.
  • El operador OR (||) se utiliza para evaluar si al menos una de dos condiciones es verdadera.
  • El operador AND (&&) se utiliza para evaluar si ambas condiciones son verdaderas.
  • El operador NOT (!) se utiliza para invertir el resultado de una condición, es decir, si una condición es verdadera, NOT la hace falsa y viceversa.
